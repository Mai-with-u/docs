## 常见配置错误

### 1. 语法错误

| 错误示例 | 正确写法 | 说明 |
|----------|----------|------|
| `enabled = "true"` | `enabled = true` | 布尔值不加引号 |
| `max_context_length = "20"` | `max_context_length = 20` | 数字不加引号 |
| 缺少逗号 | 添加逗号 | TOML 数组元素间需要逗号 |

### 2. 引用错误

| 错误示例 | 正确写法 | 说明 |
|----------|----------|------|
| `model_list = ["gpt-5"]` | `model_list = ["gpt-3.5-turbo"]` | 引用未定义的模型 |
| `api_provider = "openaai"` | `api_provider = "openai"` | 拼写错误 |
| `platform = "qq:123456789"` | `platform = "qq"` | 格式错误 |

### 3. 格式错误

| 错误示例 | 正确写法 | 说明 |
|----------|----------|------|
| `time = "9:00-18:00"` | `time = "09:00-18:00"` | 时间需要两位数字 |
| `chat_stream_id = "qq:123456"` | `chat_stream_id = "qq:123456:group"` | 需要完整格式 |
| `learning_list = [["", "enable"]]` | `learning_list = [["", "enable", "enable", "enable"]]` | 需要四位数组 |

# 📱 支持哪些平台

MaiBot 可以连接到各种聊天平台，就像万能转换器一样！

## 目前支持的平台

### 🐧 QQ（最推荐）
通过 NapCat 连接 QQ，功能最完整：
- ✅ 文字聊天
- ✅ 表情包
- ✅ 图片消息
- ✅ 群聊和私聊
- ✅ 文件传输
- ✅ 支持插件模式（推荐，更稳定更简单）

**设置简单**：插件模式一键安装，独立模式装好 NapCat 填个 IP 就能用

## 其他适配器

### GoCQ 适配器
基于 go-cqhttp 的 QQ 适配器，适合部分风控较低的账号使用：
- [GoCQ 适配器文档](./gocq)

### 社区第三方适配器
社区开发的其他平台适配器，包括微信、Discord、Telegram 等：
- 关注 [MaiBot GitHub 组织](https://github.com/Mai-with-u) 获取最新适配器
- 在社区交流群中获取第三方适配器信息

## 怎么连接平台？

### 第一步：选择适配器
每个平台都有对应的"适配器"：
- QQ → NapCat 适配器（支持插件模式，推荐！）
- 其他平台 → 社区适配器（微信、Discord 等）

### 第二步：配置连接
在配置文件里填写：
- 平台类型
- 连接地址
- 账号信息

### 第三步：启动连接
保存配置，重启 MaiBot，就自动连接了！

## 使用建议

### 新手推荐
**先用 QQ**，推荐使用插件模式：
- 设置最简单
- 文档最完善
- 社区支持最好

### 进阶玩法
**多平台联动**：
- QQ 聊天为主
- 结合社区适配器扩展更多平台

### 多平台部署
可以同时连接多个平台：
- 一个 MaiBot 服务多个群
- 不同平台不同性格
- 统一管理更方便

## 常见问题

**Q: 一定要用 QQ 吗？**
A: 目前 QQ 支持最好，建议先用 QQ 入门

**Q: 微信什么时候能用？**
A: 正在开发，可以关注更新动态

**Q: 能同时用多个平台吗？**
A: 可以，一个 MaiBot 可以服务多个平台

**Q: 平台会封号吗？**
A: 任何平台都有风险，建议用小号，遵守平台规则
